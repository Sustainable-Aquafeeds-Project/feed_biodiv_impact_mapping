# feed_biodiv_impact_mapping
Mapping biodiversity impacts for the Sustainable Aquafeeds Project


Folders: 

 - /data: Contains raw and tidied data that is used in the prep scripts.
 - /prep: Contains prep scripts for tidying data. 
 - /src: Contains functions that read in file paths and necessary directories.
 - /anaysis: Contains scripts that overlay raw material ingredient origins and species area of habitats, and conducts formal analysis.
